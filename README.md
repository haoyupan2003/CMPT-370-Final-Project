# CMPT-370-Final-Project: 3D Dice Roller

An interactive 3D dice roller application that simulates the experience of rolling physical dice in a virtual environment using WebGL 2.0.

## 1. Motivation and Purpose

The "3D Dice Roller" provides users with a realistic and visually appealing way to roll dice for games, decision-making, or educational purposes. The application supports various types of dice (D4, D6, D8, D10, D12, D20), making it suitable for tabletop games, RPGs, and other applications.

### Key Objectives

- Demonstrate WebGL 2.0 capabilities for 3D rendering, animations, and textures
- Provide an engaging and interactive user experience
- Support multiple dice types with add/remove functionality
- Implement realistic physics-based rolling animations
- Showcase originality through custom textures and interactions

## 2. Research and Technologies

### Similar Projects Analyzed

1. **Google Built-in Dice Roller**
   - Simple 2D dice roller in Google Search
   - Supports up to five D6 dice
   - Reference: [Google Dice Roller](https://g.co/kgs/nUUjMi7)

2. **DiceParser**
   - JavaScript library for dice notation parsing
   - Backend logic for roll simulation
   - Reference: [DiceParser GitHub](https://github.com/GreenImp/rpg-dice-roller)

3. **Dice Simulator**
   - 3D dice roll simulator with physics
   - Suitable for D&D and board games
   - Reference: [Dice Simulator](https://dice-simulator.com/)

4. **3D Die Roller by Emanuele Marchi**
   - Interactive 3D interface
   - Supports multiple dice types
   - Reference: [3D Die Roller](https://emanchado.github.io/3d-die-roller/dice/)

### Technologies Used

- WebGL 2.0 for 3D graphics rendering
- GLSL for shader programming
- HTML5 and JavaScript for UI
- Blender for 3D modeling
- Custom physics implementation

## 3. Project Features

### Core Functionality

- Multiple dice types (D4, D6, D8, D10, D12, D20)
- Realistic rolling animations with physics
- Custom textures and 3D models
- Interactive user controls
- Result tracking and display

### Technical Components

#### 3D Elements
- Custom dice models with rounded edges
- Textured faces with numbers/dots
- Physics-based animations
- Realistic lighting and shadows

#### User Interface
- Add/remove dice controls
- Dice type selection
- Roll button
- Result tooltips
- Camera controls

#### Interactions
- Mouse-based rolling
- Dice type switching
- Camera manipulation
- Result viewing
